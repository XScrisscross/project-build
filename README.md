## 说明

- 支持 VUE2/VUE3/REACT
- 支持 WEB/MIRCO-WEB/ELECTRON

## 环境

- npm run install

## 启动

#### 启动vue3
- cd ./project
- cd ./sub-vue3-template
- npm run dev 或指令

#### 启动vue 常规 webapp
- cd ./project
- cd ./sub-vue-template
- npm run serve:devp 或指令

#### 启动vue electron webapp
- cd ./project
- cd ./sub-vue-template
- npm run electron:devp 或指令
