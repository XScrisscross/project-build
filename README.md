## 说明
- 支持 VUE/VUE3/REACT
- 支持 WEB/QIANKUN-MIRCO-WEB/ELECTRON

## 镜像
- npm config set registry https://registry.npmmirror.com
- npm config set ELECTRON_MIRROR https://npmmirror.com/mirrors/electron/
- npm config set registry https://mirrors.huaweicloud.com/repository/npm/
- npm config set electron_mirror https://mirrors.huaweicloud.com/electron/

## 环境
- npm i

## 启动
#### 启动vue3
- cd .\projects\
- cd .\sub-vue3-template\
- npm run dev

#### 启动vue 常规 webapp
- cd .\projects\
- cd .\sub-vue-template\
- npm run serve:devp

#### 启动vue electron webapp
- cd .\projects\
- cd .\sub-vue-template\
- npm run electron:devp

#### 启动react 常规 webapp
- cd .\projects\
- cd .\sub-react-template\
- npm run start

#### 启动micro-app (protal-subs、sub-vue-template)
- cd .\projects\
- npm run start
